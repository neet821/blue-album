# 🔧 问题修复 - 路径别名配置

## ❌ 遇到的问题
```
Failed to resolve import "@/stores/auth" from "LinkDashboard.vue"
```

## ✅ 解决方案
在 `vite.config.js` 中添加了路径别名配置:

```javascript
import path from 'path'

export default defineConfig({
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
})
```
